# Changelog

All notable changes to Hermes: Godspeed are documented here. The project follows a static-web-build release model; the npm package is intentionally marked `private` and is not published. See [RELEASE.md](RELEASE.md) for the release process.

## 0.1.0

Initial public release pass of Hermes: Godspeed.

- Lane-based running, jumping, and sliding with keyboard and touch controls.
- Lobster collecting, combo scoring, obstacles, and divine powerups (Chomp, Sandals, Goblet, Zeus Bolt, Lobster Magnet).
- Phase-based spawn variety, route segments, and Herald Gates with boons.
- Classic and Daily Dispatch run modes.
- Local high score persistence per difficulty and local mission progression, laurels, and cosmetic unlocks.
- Procedural Web Audio music and sound effects.
- Optional in-browser MP4 run recording when supported by the browser.
- First-run intro cutscene with optional narration.
- Release-bundle assertion that rejects shipped test/debug hooks.
